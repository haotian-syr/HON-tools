hypergraph

DOI: 10.18129/B9.bioc.hypergraph    

This package is for version 3.15 of Bioconductor; for the stable, up-to-date release version, see hypergraph.

A package providing hypergraph data structures

Bioconductor version: 3.15

A package that implements some simple capabilities for representing and manipulating hypergraphs.

Author: Seth Falcon, Robert Gentleman

Maintainer: Bioconductor Package Maintainer <maintainer at bioconductor.org>

Citation (from within R, enter citation("hypergraph")):

Installation

To install this package, start R (version "4.2") and enter: 

if (!require("BiocManager", quietly = TRUE))
    install.packages("BiocManager")

BiocManager::install("hypergraph")

For older versions of R, please refer to the appropriate Bioconductor release.
